I made this for my job interview at Bizzabo!

For my final interview with Bizzabo I was tasked with making an event calendar that called the Bizzabo API. I successfully delivered this project over the course of a weekend and am really happy with the result.
I lost API access but I have filled this with dummy data to give an idea of how it works! Essentially I just get JSON from an API and implement the events into the functionality of the Angular Calendar made by mattlewis92 on Github.

This programmatically adds events to this nifty little calendar! :)
